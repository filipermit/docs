# permit.api.users.list()

Fetch a list of the available users.

### Parameters

`page` - The page number of the results to fetch, starting at 1.<br/>
`perPage` - The number of results per page (default is 100).<br/>

### Implementation

```java
import io.permit.sdk.openapi.models.PaginatedResultUserRead;

PaginatedResultUserRead users = permit.api.users.list();
```
