# permit.api.users.unassignRole()

Unassign a role from a specific user.

### Parameters

`user` - The user the role will be unassigned to (accepts either the user id or the user key).<br/>
`role` - The role that will be unassigned (accepts either the role id or the role key).<br/>
`tenant` - The tenant the role is unassociated with (accepts either the tenant id or the tenant key).<br/>

### Implementation

```java
permit.api.users.unassignRole("elon@tesla.com", "admin", "tesla");
```
