---
title: Resource not found
---
You got this error because we can't find the resource that you pass in the permit check. Please make sure you pass the right resource in the permit check(case sensitive).

### How to fix it
#### Create resource with Permit web application
You can create resource with our [Policy management screen](https://app.permit.io/policy-editor) 

#### Create resource with the API
You can create resource with our [Resource management API](https://api.permit.io/v2/redoc#tag/Resources/operation/create_resource)

#### Create resource with the SDK
You can create resource with our [Python SDK](https://docs.permit.io/sdk/python/resource/create-resource), [Node SDK](https://docs.permit.io/sdk/nodejs/resource/create-resource) and [more](https://docs.permit.io/)
