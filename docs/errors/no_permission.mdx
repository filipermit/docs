---
title: No permission found for user to this action on this resource
---
You got this error because you'r user doesn't have the right role to perform this action or you need to add those permission to the role that you'r user has.

### How to fix it
#### Assign permission to user roles with Permit web application
You can assign this permission to user role with the [Policy management screen](https://app.permit.io/policy-editor) of the Permit web application.
Check the right action on the right resource and click the save to propegate the change to the PDP.
#### Assign permission to user roles with the API
You can use the API to assign this permission to user role. [redoc](https://api.permit.io/v2/redoc#tag/Role-Assignments/operation/assign_role)
#### Assign permission to user roles with the SDK
You can use the SDK to assign this permission to user role. [Python SDK](https://docs.permit.io/sdk/python/role/assign-role), [Node SDK](https://docs.permit.io/sdk/nodejs/role/assign-role) and [more](https://docs.permit.io/)
