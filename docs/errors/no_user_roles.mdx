---
title: No roles found for user
---
You got this error because you are trying to check permission for a user that don't have any roles assignment.

### How to fix it
#### Assign roles to user with Permit web application
You can assign roles to user with our [Users management screen](https://app.permit.io/user-management) of the Permit web application and make sure you assign the right role on the same tenant that you pass in the permit check.
#### Assign roles to user with the API
You can use the API to assign roles to user. [redoc](https://api.permit.io/v2/redoc#tag/Role-Assignments/operation/assign_role)
#### Assign roles to user with the SDK
You can use the SDK to assign roles to user. [Python SDK](https://docs.permit.io/sdk/python/role/assign-role), [Node SDK](https://docs.permit.io/sdk/nodejs/role/assign-role) and [more](https://docs.permit.io/)
