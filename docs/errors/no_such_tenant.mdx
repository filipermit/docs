---
title: Tenant not found
---
You got this error because we can't find the tenant that you pass in the permit check. Please make sure you pass the right tenant in the permit check(case sensitive).

### How to fix it
#### Create tenant with user and roles with Permit web application
You can create tenant with our [Tenant management screen](https://app.permit.io/user-management/tenants) 
Than create user and create user and roles with our [User management screen](https://app.permit.io/user-management/users) on this tenant.

#### Create tenant with user and roles with the API
You can create tenant with our [Tenant management API](https://api.permit.io/v2/redoc#tag/Tenants/operation/create_tenant)
You can use the API to create user and roles inside this tenant. [create user](https://api.permit.io/v2/redoc#tag/Users/operation/create_user)[assign role](https://api.permit.io/v2/redoc#tag/Role-Assignments/operation/assign_role)
#### Create tenant with user and roles with the SDK
You can create tenant with our [Python SDK](https://docs.permit.io/sdk/python/tenant/create-tenant), [Node SDK](https://docs.permit.io/sdk/nodejs/tenant/create-tenant) and [more](https://docs.permit.io/)
You can use the SDK to create user and roles inside this tenant.
